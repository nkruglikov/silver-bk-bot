Маршруты хранятся в папке routes с расширением .txt.
Файл маршрута должен состоять из списка команд, по одной команде на строку.
Если команда поддерживает параметры, они должны быть указаны через двоеточие.
Пример: КОМАНДА:ПАРАМЕТР1:ПАРАМЕТР2
В параметрах допускаются пробелы. Начальные и конечные пробелы будут отрезаны.
Например, следующие команды эквивалентны:
КОМАНДА:ДЛИННЫЙ ПАРАМЕТР1:ПАРАМЕТР2
 КОМАНДА :  ДЛИННЫЙ ПАРАМЕТР1  : ПАРАМЕТР2

Доступые команды:

A. Команды для Портала воспоминаний.

ЗАДАНИЕ:Имя пещеры -- получить задание
НАГРАДА:Имя пещеры -- получить награду
НАЧАТЬ:Имя пещеры  -- войти в пещеру

B. Команды для пещеры
ВПЕРЕД, НАЗАД, ВПРАВО, ВЛЕВО -- перемещения
НАПРАВО, НАЛЕВО -- повороты
ПОДНЯТЬ -- поднять предметы
ИСПОЛЬЗОВАТЬ:Имя объекты -- использовать объект. Имя указывать обязательно!
НАПАСТЬ -- Атаковать первого монстра из доступных
ВЫЗДОРОВЕТЬ -- ждать до полного восстановления здоровья

C. Команды для режима атаки
УДАРИТЬ:АТАКА:ЗАЩИТА -- Выбрать атаку и защиту на радиобаттонах и ударить.
                        Параметры можно не указывать, тогда выберутся первые
                        значения.
                        Радиобаттоны нумеруются с нуля.
ПРИЕМ:НОМЕР1:НОМЕР2:... -- Применить приёмы с номерами НОМЕР1, НОМЕР2, ...
                           Если применить без параметров -- ничего не выполнится.
ЗАКОНЧИТЬ -- Нажать на кнопку "Вернуться"

D. Дополнительные команды
БОЙ:АТАКА:ЗАЩИТА:НОМЕР1:НОМЕР2:... -- выполнить команды в следующем порядке:
                                      НАПАСТЬ
                                      если есть кнопка "Вернуться" -- ЗАКОНЧИТЬ
                                      ПРИЕМ:НОМЕР1:НОМЕР2:...
                                      УДАРИТЬ:АТАКА:ЗАЩИТА
                                      Повторить.
ЖДАТЬ:N -- Ждать N секунд

E. Системные команды
%Маршрут -- Пройти по маршруту с именем "Маршрут.txt".
КОНСОЛЬ -- Читать команды, введённые пользователем в консоль вручную,
           до команды ВЫХОД.
ВЫХОД -- покинуть режим консоли.


Файл interpreter.exe выводит список маршрутов, найденных в папке routes,
и выполняет команду КОНСОЛЬ.
Для корректной работы необходимо, чтобы был установлен браузер Chrome.
Окно браузера, открывающееся после запуска программы, нельзя сворачивать,
но оно не обязано быть в фокусе -- можно перекрывать его другими окнами.
Курсор мыши не захватывается.
